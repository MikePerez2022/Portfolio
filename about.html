About Me

I’m a developer passionate about both software and game development, with a strong interest in dynamic systems, AI, and emerging technologies.

In Hop n' Hook, a fast-paced 2D platformer, I was responsible for the majority of the user interface design and implementation, including animated UI elements, customizable keybinding, and a save/load system. The project was built in Unity using C# and focused on responsive gameplay and clean UX.

For AI Story Generator, I developed a desktop application that uses AI to generate short stories based on user-defined prompts, genres, and tones. Built with Python, Tkinter, and the Mistral 7B model via Ollama, the app features a modern interface and allows exporting the generated stories.

I also created AI Code Commenter, a desktop app that uses the CodeLlama 7B model via Ollama to generate intelligent, context-aware comments for code. Built with Python and Tkinter, the app focuses on enhancing code readability and maintainability through a clean and user-friendly interface.

I'm always eager to collaborate on projects that are meaningful, challenging, and innovative—whether in the world of interactive games or impactful software solutions.

Skills: Unreal Engine, Unity, C++, C#, Python, JavaScript, React.js, Git, Blueprints, AI Systems
